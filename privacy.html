개인정보처리방침

서비스명: Luvit (러빗)
운영자: East Noise
최종 업데이트: 2026년 5월 24일
시행일: 2026년 5월 24일

East Noise(이하 "회사")는 Luvit 서비스(이하 "서비스") 이용자의 개인정보를 소중히 다루며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 방침은 회사가 수집·이용·보관·파기하는 개인정보의 항목과 처리 절차를 안내합니다.

1. 수집하는 개인정보 항목

가. 회원가입 및 인증 (Google/Apple 로그인)

  • 이메일 주소
  • 이름 또는 닉네임 (계정 제공값)
  • 프로필 사진 URL (선택)
  • Apple/Google 사용자 고유 식별자 (uid)

나. 서비스 이용 과정에서 자동 수집

  • 디바이스 식별자 (Firebase Cloud Messaging 푸시 토큰)
  • 앱 사용 기록 (미션 수행, 추측, 판정, 출석)
  • 크래시 및 성능 진단 데이터 (Firebase Crashlytics)

다. 사용자가 직접 입력

  • 파트너 별명
  • 미션 추측 답변
  • 캐릭터 선택 (블루 / 핑크)
  • 커플이 처음 만난 날짜 (선택)

2. 개인정보 이용 목적

  • 회원 인증 및 본인 확인
  • 커플 연결 및 매칭
  • 데일리 미션 발급·판정·정산
  • 추측 결과·알림 푸시 발송
  • 서비스 개선을 위한 통계 분석
  • 오류 추적 및 디버깅

3. 개인정보 보유 및 이용 기간

회원 탈퇴 시 즉시 모든 개인정보를 영구 삭제합니다. 다만 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 별도 보관합니다(예: 통신비밀보호법에 따른 접속 로그 3개월).

4. 개인정보 제3자 제공

회사는 사용자 개인정보를 제3자에게 제공하지 않습니다. 다만 다음의 처리위탁이 있습니다.

  • 위탁받는 자: Google LLC (Firebase)
  • 위탁 업무: 인증, 데이터 저장, 푸시 알림, 크래시 분석
  • 보유 기간: 회원 탈퇴 시까지

5. 개인정보 파기 절차

회원 탈퇴 시 앱 내 'deleteAccount' 흐름이 다음 작업을 즉시 수행합니다.

  • Firestore users 문서 삭제
  • 모든 pointLogs 삭제
  • 커플 연결 정보 정리
  • Firebase Auth 계정 삭제

삭제된 데이터는 복구 불가능합니다.

6. 사용자의 권리

  • 본인 정보 열람: 앱 내 '설정' 메뉴
  • 정보 수정: 앱 내 '닉네임/캐릭터' 메뉴
  • 회원 탈퇴 (모든 정보 삭제): 앱 내 '회원 탈퇴' 메뉴

7. 안전성 확보 조치

  • 모든 통신 HTTPS/TLS 암호화
  • Firebase 보안 규칙로 접근 제어 (본인 또는 파트너만 데이터 접근)
  • 데이터 변경은 모두 서버 Cloud Functions 경유 (클라이언트 직접 쓰기 차단)

8. 광고 및 트래킹

현재 서비스에서는 광고를 표시하지 않으며 광고 식별자를 사용하지 않습니다. 향후 광고를 도입하는 경우 본 방침을 갱신하고 사용자에게 별도 동의를 받습니다.

9. 만 14세 미만 아동의 개인정보

회사는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

10. 정책 변경 고지

본 방침이 변경되는 경우 앱 내 공지 또는 본 페이지를 통해 사전 안내합니다.
